# Récap des commandes utiles pour votre modération


## COMMANDES DIFFUSEUR :

Pour mettre modérateur une personne :
```
/mod <user>
```       
Pour retirer un modérateur :

```
/unmod <user>
```     
Pour donner le statut VIP à quelqu’un :

```
/vip <user>
```       
Retire le statut VIP :

```
/unvip <user>
```
